Azure CLI deployment variants

Variant 1. Manual integration from GitHub

Set yours values for the next variables in the deployment/github.ps1 script:

  1. $SubID – subscription I.D.;
  2. $ResGrpNomini – resource group name;
  3. $AppSvcPlanNomini – App Service plan name;

You may also want to change another variables ($WebAppNomini and $GHRepoURL) but script should work without it smoothly.

Run the deployment/github.ps1 script. It will create the the app, configure deployment from GitHub with manual integration. Azure CLI must be installed locally in order to use the script.
