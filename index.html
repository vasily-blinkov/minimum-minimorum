Azure CLI deployment

To configure deployment of this site to Azure App Service as a web app (assuming a resource group and a service plan exist): 

            az webapp create --name MinimumMinimorum --plan "/subscriptions/0efe8e0b-fdce-4064-a660-008339591626/resourceGroups/SwallowTheDictionary-rg/providers/Microsoft.Web/serverfarms/SwallowTheDictionary-plan" --resource-group "SwallowTheDictionary-rg" --deployment-source-url https://github.com/vasily-blinkov/minimum-minimorum
